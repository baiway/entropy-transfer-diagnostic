# entropy-transfer-diagnostic
Electrostatic simulation using Tobias's diagnostic: [cbc-electrostatic-kappa1p5](cbc-electrostatic-kappa1p5). Everything is here except the `.out.nc` file, which can be found [here](https://drive.google.com/drive/folders/1theL3d0pSAa5-fc1Z9jMCEpcLqNPkA54?usp=sharing) (this was uploaded to a Google Drive separately as it's 20 GB). Input parameters from E. Belli (2008). _Effects of plasma shaping on nonlinear gyrokinetic turbulence_. (https://doi.org/10.1063/1.2972160)[doi].

Jupyter Notebook with analysis: [data-analysis/electrostatic-analysis.ipynb](data-analysis/electrostatic-analysis.ipynb).
